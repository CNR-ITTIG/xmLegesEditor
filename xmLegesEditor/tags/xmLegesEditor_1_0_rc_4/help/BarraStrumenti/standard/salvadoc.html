Salvataggio di un documento

  1. Eseguire una delle seguenti operazioni:

    • Scegliere Salva dal menù File;
    • Fare clic sul pulsante Salva sulla barra degli strumenti Standard;

  2. Se il documento viene salvato per la prima volta, compare la finestra di esplorazione delle cartelle:

    - Dalla finestra di esplorazione delle cartelle, selezionare la cartella nella quale si vuol salvare il file e aprirla;
    - Nella casella Nome file digitare il nome del documento, il file viene salvato con estensione xml
    (È possibile utilizzare nomi di file lunghi e descrittivi. Per i nomi di file non è possibile utilizzare i seguenti caratteri: barra (/), barra rovesciata (\), segno di maggiore (>), segno di minore (<), asterisco (*), punto interrogativo (?), virgolette ("), linea verticale (|), due punti (:) o punto e virgola (;)).

    Se il documento era già stato salvato precedentemente, viene sostituita la vecchia versione del documento con la nuova corrente.
