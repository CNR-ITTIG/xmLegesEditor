Importazione in un documento di una struttura XML-NIR esterna

  1. Selezionare Import DOM dal menù Modifica;
    Nella finestra Documento corrente compare la struttura del documento che si vuol modificare;
  2. Fare clic sul pulsante Apri ;
  3. Se si desidera aprire un documento salvato in una cartella diversa, trovare la cartella e aprirla.
    Fare doppio clic sul documento che si desidera aprire, oppure selezionare il file e fare clic su Apri.
    Se il documento desiderato non è fra quelli visualizzati nell'elenco delle cartelle, è possibile effettuare una ricerca.
    Nella finestra Sorgente compare la struttura del documento dal quale si vuol importare una struttura DOM;
  4. Fare clic su un elemento della struttura del documento Sorgente;
  5. Trascinare l'elemento all'interno di un elemento del Documento corrente (l'operazione di importazione (copia) avrà effetto solo se la posizione scelta per l'elemento è ammessa dalla DTD del Documento corrente);

Nota: DOM (Document Object Model) è il nome della rappresentazione della struttura di un documento XML.
