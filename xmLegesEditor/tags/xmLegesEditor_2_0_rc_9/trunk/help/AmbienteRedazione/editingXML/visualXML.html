Visualizzazione della struttura di un documento NIR

La struttura di un documento aperto può essere visualizzata nella finestra Struttura XML a sinistra di quella principale di editing.

Nella finestra Struttura XML posizionarsi su un elemento XML-NIR

  1. Fare clic sui rami della struttura ad albero per espanderne i nodi (elementi XML-NIR);
  2. Fare clic sul nodo che si vuole selezionare (gli attributi dell'elemento vengono visualizzati nella finestra degli attributi sottostante).

Nota tecnica: Con queste operazioni è possibile visualizzare l'intera struttura XML del documento NIR (ovvero l'albero DOM (Document Object Model)) che lo rappresenta, nonché gli attributi associati a ciascun elemento.

 

"Drag and drop" di elementi della struttura XML-NIR

  1. Dalla visualizzazione struttura di un documento, fare clic sull'elemento che si desidera spostare;
  2. Trascinare la selezione sulla visualizzazione struttura nella posizione desiderata; l'operazione avrà successo solo se la posizione scelta è ammessa dalle regole delle DTD-NIR.
