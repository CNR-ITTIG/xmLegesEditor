Inserisci Immagine

  1. Selezionare il pulsante dal menù inserisci o dalla barra inserisci;
  2. aperta la maschera di inserimento:
    • nella casella Src va inserita l'origine del file (si può usare il tasto FileBrowser);
    • nella casella Alt inserire il testo alternativo all'immagine;
    • nella casella Width inserire l'ampiezza dell'immagine scegliendo fra px (pixel) o % (percentuale);
    • nella casella Height inserire l'altezza dell'immagine scegliendo fra px (pixel) o % (percentuale);
  3. Premere il bottone OK per confermare l'inserimento;
