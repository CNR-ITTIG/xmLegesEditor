Marcatura dell'annesso Tabella

La Tabella può essere inserita secondo le tre modalità sotto indicate.

Documento vuoto (in questo caso viene proposto un documento da riempire):

1. cliccare dalla Barra degli  strumenti sull'icona   Inserisci un nuovo Annesso;
2. scegliere nel Message Box Annesso l'opzione Documento vuoto;
3. riempire i campi sottostanti che appaiono nel Message Box (Denominazione e Nome file sono sempre obbligatori); Denominazione sarà in genere "Tabella";
4. accanto a Nome file, cliccare sull'icona   ;
5. nel Message Box Nuovo  selezionare in Categorie provvedimenti la voce "Generici" e in Provvedimenti la voce “Tabella” e cliccare su Ok;
5. cliccare su Ok del Message Box Annesso;
6. inserire/eliminare le righe e/o le colonne utilizzando i pulsanti della Barra degli strumenti ,, ,, ,, , ;
7. copiare sul testo la tabella che appare, ovvero il contenuto originario della tabella nelle celle corrispondenti.

Includi file (in questo caso viene richiesto il nome del file da annettere):

1. cliccare dalla Barra degli strumenti sull'icona   Inserisci un nuovo Annesso;
2. scegliere nel Message Box Annesso l'opzione Includi file;
3. riempire i campi sottostanti che appaiono nel Message Box (Denominazione e Nome file sono sempre obbligatori); Denominazione sarà in genere “Tabella”;
4. accanto a Nome file cliccando sull'icona  ;
5. scegliere il file XML dell'annesso da inserire e cliccare su Ok;
5. cliccare su Ok del Message Box Annesso.

Riferimento (in questo caso viene richiesto di inserire la URN del documento da annettere):

1. cliccare dalla Barra degli strumenti sull'icona   Inserisci un nuovo Annesso;
2. scegliere nel Message Box Annesso l'opzione Riferimento;
3. riempire i campi sottostanti che appaiono nel Message Box (Denominazione e Nome file sono sempre obbligatori); Denominazione sarà in genere “Tabella”;
4. cliccare su Ok del Message Box Annesso;
5. inserire la URN del documento da annettere (vedi argomento).
