Salvataggio di una copia di un documento

    Eseguire le seguenti operazioni:

    1. aprire il documento di cui si desidera creare una copia;
    2. scegliere Salva con nome dal menù File;
    3. se si desidera salvare il documento in una cartella diversa, trovare la cartella e aprirla;
    4. nella casella Nome file digitare il nuovo nome del documento (è possibile utilizzare nomi di file lunghi e descrittivi. Per i nomi di file non è possibile utilizzare i seguenti caratteri: barra (/), barra rovesciata (\), segno di maggiore (>), segno di minore (<), asterisco (*), punto interrogativo (?), virgolette ("), linea verticale (|), due punti (:) o punto e virgola (;));
    5. scegliere il pulsante Salva.
