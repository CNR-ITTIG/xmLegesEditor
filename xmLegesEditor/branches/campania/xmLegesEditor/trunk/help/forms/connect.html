Apri connessione remota

  1. Selezionare Connetti dal menù file e successivamente scegliere fra server FTP/WebDAV e operazione di LETTURA/SCRITTURA.
  2. Effettuare il login al server remoto:
    • nella casella Host va inserito l'indirizzo del server remoto;
    • nella casella User e Password le relative credenziali di accesso (se necessarie);
    • premere l'icona per effettuare la connessione;
  3. Selezionare un path remoto e successivamente:
    • scegliere un file remoto, per aprirlo, selezionanto OK;
    • scrivere un nome, per salvarlo in remoto, selezionanto OK
