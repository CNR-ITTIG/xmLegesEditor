Bug Report

  1. Selezionare il pulsante Segnalazione Bug dal menù strumenti o dal pannello Segnalazioni;
  2. aperta la maschera di segnalazione dei bug:
    • nella casella Mittente inserire la vostra email;
    • nella casella Server-SMTP inserire il nome del server SMTP;
    • nella casella Oggetto inserire l'oggetto della mail;
    • nella casella Descrizione inserire la descrizione delle operazioni effettuate per ricreare l'errore, possibilmente indicando anche il sistema operativo utilizzato e la versione dell'editore (indicata nel menù Aiuto>Informazioni su...);
  3. Premere il bottone Invia per inviare la mail;
  4. Premere il bottone Azzera per cancellare la casella di testo sottostante contenente l'errore;
